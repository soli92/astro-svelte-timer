# ⏱️ Astro Svelte Timer

Un timer interattivo e simpatico costruito con **Astro 6** e **Svelte 5**.

## ✨ Features

- ▶️ Start / ⏸ Pausa / 🔄 Reset
- 🏁 Lap times con confetti animati
- 😴→🤯 Mood emoji che cambia in base al tempo trascorso
- 🟡 Anello di progresso animato (ogni minuto)
- 💜 Design glassmorphism su sfondo gradiente
- 📱 Completamente responsive

## 🚀 Come avviarlo

```bash
npm install
npm run dev
```

## 🏗️ Stack

- [Astro 6](https://astro.build/)
- [Svelte 5](https://svelte.dev/)
- CSS custom (glassmorphism + animazioni)
